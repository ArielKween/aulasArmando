SGCM

Sistema de Gerenciamento de Cultivo de Mandioca

  • Objetivo Geral

    O objetivo principal do projeto é desenvolver um sistema que possa gerenciar compras e vendas de para plantações de mandioca.

  • Objetivos Específicos

  • Possibilidade de cadastro de produtos;
  • Possibilidade de cadastro de dados dos produtos;
  • Possibilidade de cadastrar a quantidade dos produtos;
  • Possibilidade de cadastro de clientes;
  • Possibilidade de cadastro de fornecedores;
  • Possibilidade do usuário determinar o quanto ele está disposto a pagar por um certo produto;
  • Inclusão de uma página principal que mostre todos os produtos disponíveis para compra;
  • Inclusão de uma página do usuário que mostre os produtos que ele está fornecendo atualmente
